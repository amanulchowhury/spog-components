SPOG Context Browser 

The context browser makes http requests to the apm endpoint https://apm-asset-perf2.run.asv-pr.ice.predix.io/v1/browse/allInstances
with parameter ?parent= to populate itself. This is done in spogContextBrowser.js with a reverse proxy. 

Instructions to set up the context browser can be found at https://devcloud.swcoe.ge.com/devspace/display/KKQTL/Context+Browser
